Axion Motors Static Website

How to use:
1. Open index.html in a browser to preview.
2. Replace info@axionmotors.com with your real email.
3. Replace images in /assets with your final photos.
4. Upload the folder to Netlify, Vercel, GitHub Pages, or any web host.

Recommended domain:
- axionmotors.com
- axionhybrid.com
- axiontrucks.com
